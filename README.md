# Bank Loan Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Bank Loan Analysis Dashboard built using Power BI. 
The goal of the project is to analyze loan data and understand loan performance, risk patterns, and customer borrowing behavior.

## Objectives
- Analyze total loan applications
- Track loan funded amounts and repayments
- Identify good loans vs bad loans
- Understand loan distribution by purpose and region

## Key KPIs
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income Ratio
- Good Loan vs Bad Loan

## Dashboard Pages

### Summary
Shows overall loan performance using KPI cards.

### Overview
Displays loan trends, loan purpose distribution, and state-wise loan analysis.

### Details
Provides detailed loan-level information for deeper analysis.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Insights
- Debt consolidation is one of the most common loan purposes.
- Most loans fall under good loan categories.
- Loan applications vary across states and months.
